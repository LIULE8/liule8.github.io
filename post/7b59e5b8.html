Java 工具类

CollectionUtils工具类

Apache Commons Collections 使操作集合时，代码更加简洁

常用API

排序：

  • 将两个已排序的集合a和b合并为一个已排序的列表，保留元素的自然顺序 CollectionUtils.collate(Iterable<? extends O> a, Iterable<? extends O> b)

  • 将两个已排序的集合a和b合并到一个已排序的列表中，根据比较器顺序 CollectionUtils.collate(Iterable<? extends O> a, Iterable<? extends O> b, Comparator<? super O> c)

  • 反转给定数组的顺序 CollectionUtils.reverseArray(Object[] array)

Null 检查：

元素不为null时，向集合添加元素：

  • CollectionUtils.addIgnoreNull(targetList, targetObject);

如果参数是null，则返回不可变的空集合（new EmptyList<>()），否则返回参数本身

  • CollectionUtils.emptyIfNull(Collection<T> collection)

空安全检查指定的集合是否为空

  • CollectionUtils.isEmpty(Collection<?> coll)
  • CollectionUtils.isNotEmpty(Collection<?> coll)

集合运算：

  • 差集：CollectionUtils.subtract(Iterable<? extends O> a, Iterable<? extends O> b)

  • 并集：CollectionUtils.union(Iterable<? extends O> a, Iterable<? extends O> b)

  • 交集：CollectionUtils.intersection(Collection a, Collection b)

  • 交集的补集：CollectionUtils.disjunction(Collection a, Collection b)

Collections

常用API

排序：

  • 正序：Collections.sort(List<T> list) 底层调用的就是 list.sort()
  • 自定义排序：Collections.sort(List<T> list, Comparator<? super T> c)
  • 倒序：Collections.reverse(List<?> list)
  • 乱序： Collections.shuffle(List<?> list)

二分查找

  • Collections.binarySearch(List<? extends Comparable<? super T>> list, T key)

交换两个下标的值

  • Collections.swap(List<?> list, int i, int j)

填充指定值

  • Collections.fill(List<? super T> list, T obj)

最小值

  • Collections.min(Collection<? extends T> coll)

最大值

  • Collections.max(Collection<? extends T> coll)

替换

  • Collections.replaceAll(List<T> list, T oldVal, T newVal)

不可修改集合

  • Collections.UnmodifiableXXX(T xxx)

加锁集合

  • Collections.synchronizedXXX(T xxx)

统计某对象出现个数

  • Collections.frequency(Collection<?> c, Object o)
