Specification 复杂查询

JPA复杂查询接口 JpaSpecificationExecutor 接口，可以完成各种复杂查询，配合 JAVA 8的新特性，使用起来特别的方便，一般用于单表的复杂查询。

注意：

  1. 针对有外键关联的表的查询条件，需要使用左联接；如上面的 employee detail name 模糊查询
  2. 如果直接 get(Employee_.detail).get(EmployeeDetail_.name) ，就是无条件内联，相当于 cross join，会产生笛卡尔积
